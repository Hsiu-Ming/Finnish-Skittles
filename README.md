# 芬蘭木棋計分王 Mölkky Master

Mölkky Master 是一個專為芬蘭木棋（Mölkky）設計的現代化計分網頁應用程式。它提供了專業的比賽管理功能，介面美觀且操作流暢，適合休閒娛樂或正式比賽使用。

## ✨ 主要功能

*   **隊伍設定**：支援自訂隊名與隊員名單，系統會自動追蹤並輪替投擲順序。
*   **專業計分板**：
    *   清晰顯示目前分數、剩餘目標分數。
    *   視覺化連續失誤指示燈（LED 風格），讓淘汰邊緣的張力倍增。
    *   清楚標示當前投擲選手。
*   **自動規則判定**：
    *   **勝利條件**：剛好達到 50 分。
    *   **爆分規則**：超過 50 分則分數退回至 25 分。
    *   **淘汰規則**：連續 3 次失誤（0分）該隊伍直接落敗。
*   **完整比賽記錄**：
    *   包含每一擲的詳細歷史記錄（回合、選手、得分、變動後分數）。
    *   支援無限次數的「復原」（Undo）功能，操作錯誤也不怕。
*   **賽後報表**：自動生成排版精美的 A4 格式比賽記錄表，支援直接列印或存成 PDF。
*   **響應式設計**：完美支援手機、平板與桌機操作，並針對手機防誤觸進行優化。

## 🛠️ 技術棧

*   **核心框架**: React 18
*   **語言**: TypeScript
*   **樣式**: Tailwind CSS (via CDN for portability)
*   **圖標**: Lucide React
*   **打包/環境**: ES Modules (無需複雜 bundler config 即可在現代瀏覽器運行)

## 🚀 如何使用

1. **設定比賽**：輸入隊伍 A 與隊伍 B 的名稱及隊員名單（每行一位），選擇先攻隊伍。
2. **開始計分**：
    *   點擊數字鍵盤輸入擊倒的分數（1-12）。
    *   若未擊倒任何木棋，請選擇紅色「失誤 (0)」按鈕。
3. **確認分數**：點擊「確認」按鈕提交該回合分數。
4. **查看記錄**：隨時點擊右上角的「記錄」按鈕查看詳細歷程。
5. **結束比賽**：當有隊伍獲勝或被淘汰時，系統會顯示勝利畫面，可選擇列印報表或重新開始。
